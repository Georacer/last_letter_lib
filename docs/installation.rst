===============================================================================
Installation
===============================================================================

Python 3.8 or higher is required to use and run this package.

Steps
===============================================================================

.. #. Clone the current repository in your machine. If you plan to use `last_letter_lib` as a submodule in your software repository, clone it whenever it fits you inside of it.
.. #. ``cd`` into its root directory.
.. #. Install python dependencies by running ``pip3 install -r requirements.txt``.
.. #. Install the project via ``pip3 install .``.

.. You are now ready to use this package. Read in the :ref:`usage` page on how to do so.
